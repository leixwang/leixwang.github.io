# 基本运算符

  • 术语
    1. 一元运算符对单一操作对象操作.如: -a, !b, i++
    2. 二元运算符操作两个操作对象.如: 2 + 3
    3. 三元运算符操作三个操作对象.如: a?b:c
  • 赋值运算符 swift的赋值操作并不返回任何值.所以下面的代码是错误的.
if x = y {
  x = 1;
  }
